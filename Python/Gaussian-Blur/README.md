## 二、高斯模糊

模糊半径外部与内部的差距非常巨大的时候，你会发现，上面这种方法的模糊效果非常的差，这种情况下，我们需要计算像素的权重，靠近中心的像素对于中心点来说关联性更强更重要，所以理应权重更高。

<br>

这种情况正好符合正态分布特点，所以选用正态分布来计算权重是一个较优的选择。
